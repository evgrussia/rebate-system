## DEC-002: Telegram Bot — полноценный ЛК

| Поле | Значение |
|------|----------|
| **ID** | DEC-002 |
| **Дата** | 2026-02-04 |
| **Контекст** | Определение роли Telegram-бота: уведомления или полноценный ЛК |
| **Решение** | Полноценный личный кабинет в Telegram-боте (не только уведомления) |

## DEC-003: Все 11 бирж в MVP

| Поле | Значение |
|------|----------|
| **ID** | DEC-003 |
| **Дата** | 2026-02-04 |
| **Контекст** | Количество бирж в MVP: начать с 3-4 или все 11 |
| **Решение** | Все 11 бирж одновременно в MVP |
| **Альтернативы** | 1) Начать с 4 (Tier 1 с API); 2) Начать с 7 (Tier 1 + 2); 3) Все 11 |

| **Стейкхолдеры** | Основатель |
| **Влияние** | Exchange Integration, Timeline (+2-4 недели), Operations |
| **Риски** | 7 бирж без affiliate API → ручной сбор данных → увеличивает операционные затраты |

## DEC-010: Trader share 30-95%, не ретроактивно

| Поле | Значение |
|------|----------|
| **ID** | DEC-010 |
| **Дата** | 2026-02-04 |
| **Контекст** | Допустимый диапазон доли трейдера и правила изменения |
| **Решение** | Min 30%, Max 95%. Изменение ставок применяется только к будущим расчётам |
