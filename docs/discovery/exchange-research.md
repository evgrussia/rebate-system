## 2. Детальный анализ по биржам

### 2.1 MEXC

**Affiliate комиссия:** 50–70% (до 70% при холде 20K MX токенов)
**Customizable sharing:** До 20% комиссии можно делить с рефералами
**Суб-аффилиаты:** 10% от комиссий суб-аффилиатов
**Длительность:** 1080 дней с момента регистрации реферала
**Выплата:** Ежедневно до 01:00 UTC
**API:** Общий trading API есть, affiliate-специфический API — через портал affiliates.mexc.com
**Сеть:** 32,000+ KOL, 170+ стран
**Особенности:** Настраиваемое разделение комиссии (сентябрь 2024), link shortener, campaign tracking

### 2.2 Bitget

**Affiliate комиссия:** До 50% (спот + фьючерсы)
**Базовая ставка:** 30% для новых аффилиатов
**On-chain affiliate:** До 40% rebate (25% прямые + 15% сетевые на 3 уровня)
**Оценка:** Ежемесячная; при несоответствии — ставка снижается до 0%
**Бонусы:** $50–$100 за нового верифицированного трейдера, milestone rewards $1K–$10K
**TradFi:** Новый канал дохода (январь 2026)
**API:** Trading API есть, affiliate dashboard доступен
**Особенности:** Строгая ежемесячная оценка (с октября 2025)

### 2.3 BingX

**Affiliate комиссия:** 40–60%
**Referral (стандарт):** 10–45% (tiered)
**Выплата:** Ежедневно, в BTC/ETH/USDT
**Broker API:** Есть! sourceKey для трекинга, endpoint `open-api.bingx.com`
**Rate limits:** 10 req/s для ордеров, 2000 req/10s для аккаунтов
**Документация:** GitHub (bingx-api), Python SDK (bingx-api PyPI)
**Переоценка:** Ежемесячно, affiliate level зависит от объёмов
**Особенности:** Broker program с API-интеграцией — отличный вариант для нашего сервиса

### 2.4 Weex

**Affiliate комиссия:** До 50–70%
**Referral (стандарт):** До 40%
**WXT токен:** До 70% trading fee rebate для холдеров
**Выплата:** T+1
**API:** **Есть affiliate API!** Endpoint для получения данных рефералов (депозиты, объёмы, комиссии)
**Auth:** ACCESS-KEY, ACCESS-SIGN, ACCESS-PASSPHRASE, ACCESS-TIMESTAMP
**Сеть:** 20,000+ аффилиатов, $300M+ выплачено
**Особенности:** Один из немногих с документированным affiliate API

### 2.5 Bybit

**Affiliate комиссия:** 30–50%
**Дополнительно:** 5% на Earn продукты, 10% override суб-аффилиатов
**Alpha Referral:** До 30% (новая программа, ноябрь 2025)
**Broker API:** Есть! OpenAPI V5, Broker ID для трекинга (`&referer=api.Abc&`)
**Особенности:** Быстрая регистрация через API (whitelist IP), cap 45% для broker + affiliate
**Выплата:** Ежедневно до 04:00 UTC, в валюте сделки
**VIP cap:** Макс 50% для VIP аккаунтов
**Документация:** bybit-exchange.github.io/docs/v5/intro

### 2.6 Binance

**Affiliate комиссия:** 41% спот + 30% фьючерсы (до 50% общий, до 80% top referrers)
**Referral Pro:** Настраиваемый rebate для рефералов
**Referral Lite:** Одноразовый $100 бонус
**KYC:** Обязательно для рефералов
**API:** Общий trading API, rebate vouchers НЕ работают через API (с сентября 2024)
**Бонусы:** До 72,000 USDT/мес futures bonus
**Особенности:** Самая большая биржа, строгие правила, высокие комиссии для top affiliates

### 2.7 Bitmart

**Affiliate комиссия:** До 100% спот / 60% фьючерсы (зависит от уровня)
**3 уровня:** Inspiration Club (500+ followers), Impact Club, Influence Club (20K+ followers)
**Суб-аффилиаты:** 10% (2 уровня)
**Referral (стандарт):** 30% Tier 1 + 10% Tier 2
**Длительность:** 1 год
**Channel Partner:** До 70% + 15% скидка для рефералов
**API:** Не обнаружен affiliate-специфический API
**Особенности:** Очень высокие ставки (до 100% спот), но для top-tier

### 2.8 OKX

**Affiliate комиссия:** До 50% (базовая 30%)
**Защитный период:** 5 месяцев (ставка не падает ниже 1-го месяца)
**Broker Program:** Есть! Tag field для трекинга, broker code в ордерах
**DEX referral:** 0–20% invitee discount, автоматические payouts on-chain
**VIP ограничения:** VIP5/6 — cap 100K/мес, VIP7+ — комиссии не начисляются
**API:** Broker API с tag system, OAuth-Broker поддержка
**Документация:** okx.com/docs-v5/broker_en/

### 2.9 KuCoin

**Affiliate комиссия:** До 60%+ (tiered, обновлено сентябрь 2025)
**Referral (стандарт):** До 35%
**Rebate для рефералов:** 20% спот / 10% фьючерсы (первые 30 дней)
**Суб-аффилиаты:** 2 уровня
**KYC:** Обязательно
**Ограничения:** Не для Бельгии, Турции, Франции, Нидерландов
**API:** Не обнаружен affiliate-специфический API
**Dashboard:** Визуализированный, данные обновляются каждые 10 минут

### 2.10 HTX (ex-Huobi)

**Affiliate комиссия:** До 60%
**Referral (стандарт):** До 20–30%
**Суб-аффилиаты:** 3 уровня
**Длительность:** 730 дней
**Rebate формула:** rebate = объём × fee × cashback %
**Расчёт:** Ежедневно, выплата до 22:00 следующего дня
**Выплата в:** USDT, point cards, или HT
**API:** Не обнаружен affiliate-специфический API
**Контакт:** [email protected] для channel-кооперации

### 2.11 BloFin

**Affiliate комиссия:** 40–50% (3 уровня: Lvl1=40%, Lvl2=45%, Lvl3=50%)
**Referral (стандарт):** 30%
**Rebate для рефералов:** До 20% (настраиваемо per ссылка)
**Суб-аффилиаты:** До 29 уровней!
**Расчёт:** Ежечасно с 00:00 UTC
**Maker/Taker ставки:** Настраиваемые per ссылка
**Фьючерсы:** 0.02% maker / 0.08% taker (non-VIP)
**API:** Не обнаружен affiliate-специфический API
**Особенности:** Очень гибкие настройки, ежечасная выплата

| Конкурент | Категория | Отличие от нас |
|-----------|-----------|---------------|

### Market Gaps (наши возможности)

1. **Нет агрегатора 11+ бирж** с единым ЛК и Telegram-ботом для конечных трейдеров
2. **Нет прозрачности** — трейдеры не видят точных данных о комиссиях и rebate
3. **Существующие rebate-сервисы** ориентированы на Forex, крипто — второстепенный канал
4. **Telegram-бот ЛК** — ни один конкурент не предлагает полноценный ЛК в Telegram

## 6. Ключевые выводы для Product/Architect

### Для Rebate Calculator:
- Каждая биржа имеет **свою формулу расчёта** — адаптер должен нормализовать данные
- Ставки **динамические** и зависят от tier-уровня — нужен мониторинг
- Некоторые биржи исключают: VIP-трейдеров, market makers, zero-fee сделки, бонусные сделки
- Валюта выплат: преимущественно USDT, но Bybit платит в валюте сделки

### Для Exchange Integration Layer:
- 4 биржи имеют **Broker/Affiliate API** — приоритет интеграции
- 7 бирж требуют **ручного/dashboard-based** подхода или контакт с AM
- Rate limits варьируются (BingX: 10 req/s, Bybit: varies)
- Необходим unified adapter pattern с per-exchange конфигурацией

### Для бизнес-модели:
- Средняя affiliate ставка: **40–50%** от комиссий трейдеров
- Если делить 50/50 с трейдером: маржа сервиса = 20–25% от комиссий
- Некоторые биржи уже поддерживают настраиваемый split (MEXC, BloFin)
