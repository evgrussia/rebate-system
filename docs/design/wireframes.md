## Общие конвенции

```
Обозначения в wireframes:

  [Кнопка]           — кликабельная кнопка (primary)
  (Кнопка)           — кнопка secondary / ghost
  [ v ]              — dropdown / select
  [_____________]    — текстовое поле ввода
  [x]                — checkbox активный
  [ ]                — checkbox неактивный
  (o)                — radio активный
  ( )                — radio неактивный
  ---                — разделитель / горизонтальная линия
  |  |               — контейнер / карточка
  {{иконка}}         — иконка
  [IMG]              — изображение / логотип
  ^^^                — навигация / breadcrumbs
  ...                — загрузка / skeleton
  #                  — заголовок уровня
```

**Сетка:** 12 колонок, max-width 1280px, gutter 24px
**Breakpoints:** Desktop 1280px+ / Tablet 768-1279px / Mobile < 768px

### Мобильная версия (< 768px)

```
Изменения:
- Форма на полную ширину с padding 16px
- Toggle Email/Telegram — full-width кнопки
- CTA "Зарегистрироваться" — fixed bottom (sticky)
- Ссылка "Войти" — под формой
```

### Аннотации

| Элемент | Поведение |
|---------|-----------|
| Toggle Email/Telegram | Переключает содержимое формы, анимация fade |
| Поле Email | Inline validation: формат email, проверка уникальности (debounce 500ms) |
| Поле Пароль | Индикатор силы пароля (weak/medium/strong) |
| Повтор пароля | Валидация совпадения при blur |
| CTA | disabled пока форма невалидна, loading spinner при отправке |
| Telegram OAuth | Redirect на TG Login Widget → callback → JWT |

### Состояния

| Состояние | Описание |
|-----------|----------|
| Default | Пустая форма, CTA disabled |
| Filling | Inline validation по мере ввода |
| Submitting | CTA показывает spinner, поля disabled |
| Success | Redirect на /dashboard, toast "Добро пожаловать!" |
| Error: Email занят | Inline ошибка под полем email + ссылка "Войти?" |
| Error: Слабый пароль | Inline ошибка + подсказка требований |
| Error: Сеть | Toast "Ошибка сети. Попробуйте снова" + кнопка Retry |
