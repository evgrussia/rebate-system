## 2. Навигационная структура (Navigation Structure)

### 2.1 Landing Page — Top Navigation

```
┌──────────────────────────────────────────────────────────────┐
│  [Logo]   Биржи   Как это работает   FAQ   [Войти] [Начать] │
└──────────────────────────────────────────────────────────────┘
```

| Позиция | Элемент | Тип | Действие |
|---------|---------|-----|----------|
| Лево | Логотип CryptoRebate | Ссылка | Переход на `/` |
| Центр | Биржи | Якорная ссылка | Скролл к секции бирж |
| Центр | Как это работает | Якорная ссылка | Скролл к секции шагов |
| Центр | FAQ | Якорная ссылка | Скролл к FAQ |
| Право | Войти | Кнопка (secondary) | Переход на `/login` |
| Право | Начать | Кнопка (primary) | Переход на `/register` |

## 3. Иерархия контента (Content Hierarchy)

### 3.1 Dashboard (Обзор) — Веб-ЛК

```
H1: Обзор
├── Карточки метрик (row, 4 шт.)

```

    └── Карточка биржи
        ├── Логотип + Название
        ├── Статус: «Подключена» / «Не подключена»

        ├── Объём: $XX,XXX

### 5.5 Правила URL

```yaml
Общие правила:
  - Все URL в нижнем регистре (lowercase)
  - Разделитель слов: дефис (-), не underscore
  - Без trailing slash
  - Параметры фильтров: query string (?exchange=mexc&period=30d)
  - Пагинация: ?page=2&per_page=20
  - ID пользователя: числовой (/admin/users/123)
  - ID выплаты: числовой (/admin/payouts/456)
  - Slug биржи: строковый (/exchanges/mexc)

Redirects:
  - "/" → "/dashboard" (если авторизован)
  - "/admin" → "/admin/dashboard"
  - "/history" → "/history/accruals"
  - "/settings" → "/settings/profile"
  - Любой protected URL → "/login" (если не авторизован)
```
